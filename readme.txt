This plugin will tie a tag to a single post or page, via the ID.

== Description ==

This is a simple plugin, that enables the admin to correlate a tag to a single post or page. The pluging will also modify the
way tag lists and clouds are generated, so the tags comes links to the post selected.

Default post ID 0 will link the tag to the actual tag permalink.

== Installation ==

1. Upload `tag2post.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
